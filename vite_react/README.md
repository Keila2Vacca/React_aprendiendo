## Tabla general de Hooks

| Hook | Descripción corta | Categoría | Autora/or |
|------|-------------------|-----------|-----------|
| `useState` | Maneja el estado local de un componente funcional | Estado | Keila |
| `useReducer` | Gestiona estado complejo mediante acciones y un reducer | Estado | |
| `useRef` | Crea una referencia mutable que no causa re-render | Referencias |  |
| `useImperativeHandle` | Expone métodos del hijo al padre vía ref | Referencias |  |
| `useContext` | Accede a un contexto sin prop drilling | Contexto y datos externos |  |
| `useSyncExternalStore` | Suscribe el componente a un store externo | Contexto y datos externos |  |
| `useId` | Genera IDs únicos y estables para accesibilidad | Contexto y datos externos |  |
| `useMemo` | Memoriza el resultado de un cálculo costoso | Performance |  |
| `useCallback` | Memoriza una función para evitar recreaciones | Performance |  |
| `useTransition` | Marca actualizaciones de estado como no urgentes | Performance |  |
| `useDeferredValue` | Difiere la actualización de un valor específico | Performance |  |
| `useEffect` | Ejecuta efectos secundarios después del render | Efectos / ciclo de vida |  |
| `useLayoutEffect` | Como useEffect pero antes de pintar en pantalla | Efectos / ciclo de vida |  |
| `useInsertionEffect` | Inyecta estilos antes de que el DOM se pinte | Efectos / ciclo de vida |  |
| `useDebugValue` | Muestra etiqueta personalizada en React DevTools | Debug |  |
| `useOptimistic` | Actualiza la UI de forma optimista antes del servidor | Nuevos en React 19 |  |
| `useFormStatus` | Accede al estado de envío del formulario padre | Nuevos en React 19 |  |
| `useActionState` | Gestiona el estado de una acción asíncrona de formulario | Nuevos en React 19 |  |
